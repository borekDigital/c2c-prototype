# Frontend
> for Change2Collect (C2C) application, powered by [Next](https://nextjs.org)

## Getting started
> If you want to start the complete project, follow the [general setup instructions](../README.md)

## Local Frontend Development

### Setup
```bash
# make command
make setup-dev
```
*or*
```bash
# yarn
yarn setup:dev
```

### Run The App
```bash
# make command
make dev
```
*or*
```bash
# yarn
yarn dev
```

## Build

### Setup
```bash
# make command
make setup-build
```
*or*
```bash
# yarn
yarn setup:build
```

### Generate The Build
```bash
# make command
make build
```
*or*
```bash
# yarn
yarn build
```

### Start The Build
```bash
# make command
make start
```
*or*
```bash
# yarn
yarn start
```
